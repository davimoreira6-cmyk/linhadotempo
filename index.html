Minha Jornada no Desenvolvimento Web

Nesta etapa do curso aprendi os principais conceitos para criar sites modernos, responsivos e interativos.

Aula 01 - Introdução ao Desenvolvimento Web

Aprendi como a internet funciona, a diferença entre front-end e back-end e a importância do desenvolvimento web.

Aula 02 - HTML Básico

Aprendi a criar a estrutura das páginas utilizando títulos, parágrafos, links, listas e imagens.

Aula 03 - Formulários HTML

Desenvolvi formulários com campos de texto, e-mail, senha, botões e seletores.

Aula 04 - CSS Básico

Personalizei páginas utilizando cores, fontes, espaçamentos e diversos estilos visuais.

Aula 05 - Box Model

Entendi o funcionamento de margin, padding, border e content na organização dos elementos.

Aula 06 - Flexbox

Aprendi a alinhar e distribuir elementos de forma flexível e organizada.

Aula 07 - Grid Layout

Utilizei Grid Layout para criar estruturas modernas com linhas e colunas.

Aula 08 - Responsividade

Aprendi a adaptar sites para celulares, tablets e computadores.

Aula 09 - Media Queries

Utilizei Media Queries para criar layouts específicos para cada tamanho de tela.

Aula 10 - Animações CSS

Criei efeitos visuais e transições para deixar os sites mais interativos.

Aula 11 - Keyframes

Desenvolvi animações personalizadas utilizando a propriedade @keyframes.

Aula 12 - Projeto Integrador

Coloquei em prática todos os conhecimentos adquiridos durante o curso em um projeto completo.

Minha Evolução Durante o Semestre

Quem eu era no início do semestre

Conhecimentos iniciais

No início do semestre eu possuía poucos conhecimentos sobre desenvolvimento web. Sabia utilizar a internet e algumas ferramentas digitais no dia a dia, mas não entendia como os sites eram construídos nem quais tecnologias eram utilizadas para criar páginas modernas. Conceitos como HTML, CSS, responsividade e animações eram praticamente desconhecidos para mim.

Expectativas

Minhas expectativas eram aprender a criar páginas simples, entender melhor o funcionamento da internet e desenvolver habilidades que pudessem ser úteis tanto nos estudos quanto em futuros projetos profissionais. Também tinha curiosidade em descobrir como os sites que utilizo diariamente são desenvolvidos.

Quem eu sou agora

Conhecimentos adquiridos

Ao longo do semestre adquiri diversos conhecimentos relacionados ao desenvolvimento web. Aprendi a estruturar páginas utilizando HTML, criar formulários para coleta de dados e personalizar layouts com CSS. Também compreendi o funcionamento do Box Model, Flexbox e Grid Layout, que são ferramentas fundamentais para organizar os elementos de uma página de forma eficiente.

Além disso, aprendi conceitos de responsividade e Media Queries, permitindo que os sites funcionem corretamente em diferentes dispositivos, como celulares, tablets e computadores. Também desenvolvi animações utilizando CSS e Keyframes, tornando as interfaces mais modernas e interativas.

Projetos realizados

Durante as aulas participei da criação de diversas páginas e exercícios práticos que me ajudaram a colocar em prática os conteúdos estudados. Cada atividade contribuiu para meu desenvolvimento e para a consolidação dos conhecimentos adquiridos ao longo do curso.

Habilidades desenvolvidas

Desenvolvi habilidades de organização, resolução de problemas, criatividade e atenção aos detalhes. Também aprendi a pesquisar soluções, interpretar códigos e criar projetos mais estruturados e profissionais.

Meu próximo objetivo

O que pretendo aprender futuramente

Meu próximo objetivo é aprofundar meus conhecimentos em desenvolvimento web aprendendo JavaScript para tornar os sites mais dinâmicos e interativos. Também pretendo estudar frameworks, banco de dados e outras tecnologias utilizadas no mercado de trabalho. Quero continuar evoluindo e criando projetos cada vez mais completos, aplicando os conhecimentos adquiridos neste semestre e desenvolvendo novas habilidades na área da tecnologia.
